This is a simple web component. Inspect element for getting the shadow DOM

To create a webcomponent

  • Create a template. const template = document.createTemplate('template');
  • Create the structure of the component. template.innerHTML = `...Here goes the style and HTML of the template`;
  • Create a class extending HTMLElement. 
    class MyComponent extends HTMLElement {
    constructor() {
        super();
        this._shadowRoot = this.attachShadow({mode: true});
        this._shadowRoot.appendChild(template.content.cloneNode(true));
    }
}
  • Then you export the class so that you can use it as HTML tags.
    window.customElements.define('my-component', MyComponent);
  • You use the component created as <my-component />
